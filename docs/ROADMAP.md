## 已移除的規劃

以下功能經評估後認為不必要或過度設計：

| 功能 | 移除原因 |
|------|----------|
| 2FA | 交友 App 不需要 |
| IP 黑名單 | 速率限制已足夠 |
| 病毒掃描 | 只上傳圖片不需要 |
| 照片自動審核 | 成本高，可人工審核 |
| SMS 通知 | 成本高，用推播代替 |
| 實名制驗證 | 成本高、用戶抗拒 |
| 限時動態 | 開發成本高 |
| 線下活動配對 | 不同產品方向 |
| 群組聊天 | 交友 App 不需要 |
| 視訊通話 | WebRTC 複雜度極高 |
| ELO 分數系統 | 複雜且效果難驗證 |
| 協同過濾推薦 | 需大量數據，早期沒用 |
| ML/AI 功能 | 需專門團隊 |
